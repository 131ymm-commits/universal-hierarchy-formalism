## Project Overview

**Universal Hierarchy Formalism (UHF)** is a unified mathematical framework that describes the evolution of systems at all scales — from quantum particles to molecules, living organisms, societies, and civilizations — using the same core principles: **Free Energy** minimization and **Information Gain**.

### Key New Development (2026)
**Cross-Level Collapse** — A new interpretation of wave function collapse.

Collapse occurs when a low-level system (e.g. quantum object) interacts with a much higher-level system (macroscopic apparatus). Large scale difference ($\Delta i \gg 0$) causes rapid coarse-graining of information, leading to loss of coherence and classical outcomes.

**Army Analogy**:  
A civilian (low level, high freedom/superposition) is drafted into the army (high level, rigid structure) → identity collapse. Demobilization → partial return to superposition with residual decoherence.
